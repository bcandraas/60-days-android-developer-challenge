# 60 Days Android Developer Comeback Challenge 🤖

This repository documents my **60-day journey to relearn modern Android development** and rebuild my portfolio after a career break.

The goal is to practice consistently, follow modern Android architecture, and build real projects using the latest Android development tools.

Started: March 2026  
Status: In Progress

---

# Learning Roadmap

The challenge is divided into several phases that gradually move from fundamentals to production-ready Android development.

---

## Phase 1 — Kotlin Foundations

**Focus:** Rebuilding Kotlin fundamentals.

Topics covered:

* Variables & functions
* Control flow
* Object-oriented programming in Kotlin
* Collections
* Null safety & error handling

Outcome:

* Kotlin practice exercises
* Small console mini-project

---

## Phase 2 — Jetpack Compose UI

**Focus:** Building modern Android UI using Jetpack Compose.

Topics covered:

* Compose layouts (Column, Row, Box)
* Modifiers & styling
* State management
* LazyColumn lists
* Navigation between screens

Outcome:

* Several UI demo screens
* Simple multi-screen Compose app

---

## Phase 3 — Android Core Fundamentals

**Focus:** Understanding essential Android components.

Topics covered:

* Activities & lifecycle
* Intents & navigation
* Resources (strings, themes, layouts)
* Basic permissions
* Android project structure

Outcome:

* Small multi-screen Android demo application

---

## Phase 4 — Modern Android Architecture

**Focus:** Asynchronous programming and MVVM architecture.

Topics covered:

* Kotlin Coroutines
* suspend functions
* Flow & StateFlow
* ViewModel
* MVVM architecture pattern

Outcome:

* MVVM structured Android app

---

## Phase 5 — Local Data Storage

**Focus:** Persisting data locally using Room Database.

Topics covered:

* Room database setup
* DAO & entity modeling
* CRUD operations
* Connecting Room with ViewModel and UI

Outcome:

* **Notes App (Portfolio Project #1)**

Features:

* Jetpack Compose UI
* MVVM architecture
* Room database
* Coroutines & StateFlow

---

## Phase 6 — Networking & APIs

**Focus:** Fetching and displaying remote data.

Topics covered:

* REST API basics
* JSON parsing
* Retrofit networking
* Error handling
* Repository pattern

Outcome:

* **API-based app (Portfolio Project #2)**
  Example: Movie app or News app

---

## Phase 7 — Clean Architecture & Dependency Injection

**Focus:** Writing scalable Android code.

Topics covered:

* Repository pattern
* Clean architecture layers
* Basic dependency injection (Hilt / Koin)
* Loading & error states
* UI polish and refactoring

Outcome:

* Refactored projects with improved architecture

---
