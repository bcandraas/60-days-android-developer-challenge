# 60 Days Android Relearning Challenge 🚀

This repository documents my **60-day journey of relearning Android development** using modern Android tools and practices.

## Goals

* Rebuild Kotlin fundamentals
* Learn modern Android UI using Jetpack Compose
* Implement MVVM architecture
* Build Android applications step by step
* Prepare for Android developer roles

---

# Tech Stack

Throughout this challenge I will practice and implement:

* Kotlin
* Jetpack Compose
* Android SDK
* MVVM Architecture
* Coroutines & Flow
* Room Database
* Retrofit (REST API)
* Dependency Injection
* Clean Architecture

# Learning Roadmap

The challenge is divided into several phases, each focusing on a specific part of the Android ecosystem.

| Phase   | Topic                       | Outcome                                         |
| ------- | --------------------------- | ----------------------------------------------- |
| Phase 1 | Kotlin Foundations          | Kotlin exercises + mini console app             |
| Phase 2 | Jetpack Compose UI          | Compose UI demos                                |
| Phase 3 | Android Core                | Multi-screen Android demo                       |
| Phase 4 | MVVM + Coroutines           | Structured Android app                          |
| Phase 5 | Local Data Storage          | Android app using Room Database                 |
| Phase 6 | Networking                  | API-based Android application                   |
| Phase 7 | Clean Architecture          | Refactored project structure                    |
| Phase 8 | Portfolio & Job Preparation | GitHub polish, CV update, interview preparation |
